Gladiator Holdings Multi-SPV Smart Contract Registry — Story Protocol & Base L2

Overview

This registry documents the smart contracts deployed by Millionaire Resilience LLC / Gladiator Holdings LLC across two EVM networks: Story Protocol Mainnet (Chain 1514) and Base L2 (Chain 8453). Contracts cover IP-backed SPV loans, attestation services, 3-of-5 multi-sig orchestration, and stablecoin escrow under Morpho Protocol. 12 contracts total across both networks.
