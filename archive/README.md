# Archive

This folder contains non-essential reference files consolidated from the repo root for cleanliness.

## Subfolders

| Folder | Contents |
|---|---|
| `notes/` | Working notes, CLI quick-references, deployment flow drafts, protocol docs |
| `flattened-contracts/` | Flattened Solidity source files (for manual verification / Etherscan) |
| `metadata-json/` | IPA metadata, attestation bridge configs, gas optimization data |
| `pdfs/` | Reference PDFs (Story Protocol docs, OpenAI exports, UCC-1 signing) |
| `legacy-configs/` | Superseded deployment guides, zip archives, duplicate package files |

These files are retained for reference but are not required for compilation, deployment, or CI/CD operations.
